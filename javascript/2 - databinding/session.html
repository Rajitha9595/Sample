About sessionStorage -

sessionStorage is part of web storage. It used to store data in browser memory for permanent but we can erase the data from browser memory.

How to create session storage? sessionStorage.setItem()

How to fetch data from session storage? sessionStorage.getItem()

How to clear data from session storage? sessionStorage.clear()
